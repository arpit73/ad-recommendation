# Project

## Running the Project

- Clone this repository
- Setup and run the frontend

  ```sh
  # In the root directory
  npm install
  npm run dev
  ```

- Setup and run the backend

  ```sh
  cd server
  pip3 install -r requirements.txt
  python3 server.py
  ```
